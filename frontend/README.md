# Frontend

웹앱 화면 개발 공간입니다. 아직 프레임워크는 초기화하지 않았습니다.

## 역할

- 사용자 화면 구현
- API 연동
- 상태 관리
- 반응형 UI 구성
- 데모 시나리오에 맞춘 사용자 흐름 구현

## 초기화 예정 항목

- 프레임워크 선택
- 패키지 매니저 선택
- 라우팅 구조 설계
- 공통 컴포넌트 구조 정의
- API 클라이언트 구성

## 예시 구조

```text
frontend/
├── src/
│   ├── components/
│   ├── pages/
│   ├── hooks/
│   ├── lib/
│   └── styles/
└── README.md
```

## 실행 방법

프레임워크 초기화 후 아래 항목을 갱신합니다.

```bash
# install

# dev

# build
```
